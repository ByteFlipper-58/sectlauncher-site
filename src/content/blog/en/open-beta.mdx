# SectLauncher

Cross‑platform Minecraft launcher with a modern UI and modular architecture.

## 📖 Overview

SectLauncher is a modern Minecraft launcher built with Clean Architecture and Kotlin Multiplatform. It aims to provide a stable, convenient experience for managing game versions, mods and servers.

### ✨ Features

**MVP:**
- 🔐 Mojang login
- 📦 Select and download Minecraft versions
- 🚀 Launch the game
- 🔄 Offline mode

**Additionally:**
- 👤 Multiple profiles
- 🔧 Mod & modpack management (CurseForge)
- 🔄 Auto‑updates for launcher and game
- 📰 News and updates from servers
- 🌐 Server support (listing, connect)

---

## 🧩 Modularity
Each source parser (Mojang API, CurseForge, Modrinth) is a separate module for easy extension and maintenance.

## 🛠️ Tech stack

- **Language**: Kotlin
- **Platform**: Kotlin Multiplatform
- **UI**: Jetpack Compose Desktop
- **Build**: Gradle
- **Architecture**: Clean Architecture
- **Testing**: Unit tests for use cases

## 🚀 Setup & Run

### Requirements
- JDK 17+
- Gradle 8.0+

### Clone
```bash
git clone https://github.com/your-username/SectLauncher.git
cd SectLauncher
```

### Build
```bash
./gradlew build
```

### Run
```bash
./gradlew run
```
